# GreatShot

Minimal, fast screenshot + annotation tool for Linux.

## Build (host)
Fedora dependencies:
```
sudo dnf install rust cargo gtk4-devel libadwaita-devel pkgconf-pkg-config gcc
```

Build and run:
```
cargo run
```
